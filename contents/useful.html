Some Useful Files


  1. Yahoo! Finance download script,
  2. Simplifying General Relativity expressions,
  3. Embedded AdS isometries,


Here I will be uploading the files that may be useful. Feel free to use, or modify them as you wish. If you manage to add documentation to some or improve them, I'd appreciate if you could share them with me and let me share them here again with others.

  • Yahoo! Finance Download Scripts for Python.

    • This provides a couple of useful functions for Y! Finance. You will probably only need the "GetStock()" function which takes a list of tickers like "['BAC']" and downloads the file for the date range, to a specified path if you like. You don't really need scipy or pandas for this function. That's for the rest of the tools that follow it. I don't remember the original links I learned this from, but it's pretty easy to write such a script. The functions should be clear, but I may add some documentation later.

  • Mathematica files for General Relativity

    • This script should help you simplify complicated GR expressions. You can provide the metric and it will automatically calculate the Christoffel connection, Riemann and Ricci tensors for you.
    From what I remember, The documentation you need is this:

    $\Gamma^\mu_{\alpha\beta}\mbox{(in GR)} \to \Gamma_{\mu,a,b}$ (in the file)

    My convention in the file was: Greek indices $\mu,\nu,...$ indicate "contravariant" (upper) indices, and Roman $a,b,...$ is used for "covariant" (lower) indices. The reason was in Mathematica there weren't any upper indices (at least to the extent that I knew when I did this), they would count as exponents. Sorry, it definitely needs documentation. I'd appreciate if anyone volunteers to do that.

  • Mathematica File for Embedded Space (like $AdS_n$ in $R^{n+1}$) Isometries

    • This one may be useful for calculating tangent space Lie algebra for a space embedded in a higher dimensional one, though the commutator definition and such should be useful elsewhere as well. I think for the Lie algebra part, I basically replaced the derivatives with a p (for example $\ro_t\to$ pt.) That's what the commutator and the "Act[ ]" function, which acts with an operator on another operator or function, uses. This one too needs a lot of documentation, but I don't remember the detail now. It's from late 2012.
